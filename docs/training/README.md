## 🎯 学习路径

### 路径1: Go开发者

```
1. OpenTelemetry基础 (2h)
   └─ 理解Traces/Metrics概念
   └─ 掌握SDK基本使用

2. 传播器实现 (2h)
   └─ 理解分布式追踪传播
   └─ 实现自定义传播器

3. 微服务可观测性实战 (4h)
   └─ 搭建完整演示平台
   └─ 实践Trace关联
```

### 路径2: 平台工程师

```
1. OpenTelemetry基础 (2h)
   └─ 快速了解概念

2. eBPF零侵入追踪 (3h)
   └─ 深入理解eBPF原理
   └─ 实现HTTP追踪器

3. 性能调优工作坊 (3h)
   └─ 持续剖析实践
   └─ 生产环境优化
```

## 📝 培训准备

### 环境要求

```bash
# Go 1.26.1+
go version

# Docker & Docker Compose
docker --version
docker-compose --version

# eBPF开发环境 (进阶课程)
sudo apt-get install llvm clang libbpf-dev
```

### 代码仓库

```bash
git clone <repository>
cd OTLP_go

# 验证环境
go test ./...
```
