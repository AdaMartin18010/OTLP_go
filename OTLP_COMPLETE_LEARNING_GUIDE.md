## 🚀 快速启动

### 环境准备

```powershell
# 1. 克隆项目
git clone <repository-url>
cd OTLP_go

# 2. 安装依赖
go mod download

# 3. 运行基础示例
.\run-app.ps1

# 4. 启动 Docker 环境
.\run-docker.ps1

# 5. 查看文档
# 打开 docs/00_index.md
```

### 第一个追踪程序

```go
package main

import (
    "context"
    "go.opentelemetry.io/otel"
    "go.opentelemetry.io/otel/trace"
)

func main() {
    // 初始化 TracerProvider
    tp := initTracer()
    defer tp.Shutdown(context.Background())
    
    // 创建 Tracer
    tracer := otel.Tracer("my-app")
    
    // 创建 Span
    ctx, span := tracer.Start(context.Background(), "main")
    defer span.End()
    
    // 执行业务逻辑
    doWork(ctx)
}
```
