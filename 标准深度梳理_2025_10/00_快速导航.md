## 🚀 5分钟快速开始

### 最小化示例

```go
package main

import (
    "context"
    "log"
    
    "go.opentelemetry.io/otel"
    "go.opentelemetry.io/otel/exporters/otlp/otlptrace/otlptracegrpc"
    "go.opentelemetry.io/otel/sdk/resource"
    sdktrace "go.opentelemetry.io/otel/sdk/trace"
    semconv "go.opentelemetry.io/otel/semconv/v1.21.0"
)

func main() {
    // 1. 创建 Exporter
    exporter, err := otlptracegrpc.New(
        context.Background(),
        otlptracegrpc.WithEndpoint("localhost:4317"),
        otlptracegrpc.WithInsecure(),
    )
    if err != nil {
        log.Fatal(err)
    }
    
    // 2. 创建 TracerProvider
    tp := sdktrace.NewTracerProvider(
        sdktrace.WithBatcher(exporter),
        sdktrace.WithResource(resource.NewWithAttributes(
            semconv.SchemaURL,
            semconv.ServiceName("my-service"),
        )),
    )
    defer tp.Shutdown(context.Background())
    
    // 3. 设置全局 TracerProvider
    otel.SetTracerProvider(tp)
    
    // 4. 创建 Span
    tracer := otel.Tracer("my-tracer")
    ctx, span := tracer.Start(context.Background(), "hello-world")
    defer span.End()
    
    // 5. 执行业务逻辑
    log.Println("Hello, OpenTelemetry!")
}
```

**详细教程**: [Provider配置指南](./04_SDK与实现/01_Tracer_Provider/01_Provider配置.md)

## 💡 最佳实践

### 快速参考

#### 1️⃣ Provider 配置最佳实践

```go
// ✅ 推荐: 完整的生产环境配置
func setupTelemetry() (*sdktrace.TracerProvider, error) {
    // Resource
    res, _ := resource.New(
        context.Background(),
        resource.WithFromEnv(),
        resource.WithHost(),
        resource.WithProcess(),
        resource.WithAttributes(
            semconv.ServiceName("my-service"),
            semconv.ServiceVersion("1.0.0"),
            semconv.DeploymentEnvironment("production"),
        ),
    )
    
    // Exporter
    exporter, _ := otlptracegrpc.New(
        context.Background(),
        otlptracegrpc.WithEndpoint("collector:4317"),
        otlptracegrpc.WithTLSCredentials(...),
        otlptracegrpc.WithCompressor("gzip"),
    )
    
    // TracerProvider
    tp := sdktrace.NewTracerProvider(
        sdktrace.WithResource(res),
        sdktrace.WithSampler(sdktrace.ParentBased(
            sdktrace.TraceIDRatioBased(0.1),
        )),
        sdktrace.WithBatcher(exporter,
            sdktrace.WithBatchTimeout(5*time.Second),
            sdktrace.WithMaxExportBatchSize(512),
            sdktrace.WithMaxQueueSize(2048),
        ),
    )
    
    otel.SetTracerProvider(tp)
    return tp, nil
}
```

**详细说明**: [Provider配置指南](./04_SDK与实现/01_Tracer_Provider/01_Provider配置.md)

## ❓ 常见问题

### 快速FAQ

<details>
<summary><b>Q1: 如何选择 gRPC 还是 HTTP 传输？</b></summary>

**推荐**:

- **gRPC**: 高性能、低延迟、内网通信（推荐）
- **HTTP**: 跨域、防火墙友好、简单调试

**详细说明**: [gRPC vs HTTP](./01_OTLP核心协议/02_gRPC传输.md#grpc-vs-http)

</details>

<details>
<summary><b>Q2: 如何配置采样策略？</b></summary>

```go
// 开发环境: 100% 采样
sdktrace.AlwaysSample()

// 生产环境: 10% 采样
sdktrace.ParentBased(sdktrace.TraceIDRatioBased(0.1))

// 高级: 自定义采样
customSampler := NewCustomSampler()
```

**详细说明**: [采样器配置](./04_SDK与实现/01_Tracer_Provider/03_采样器.md)

</details>

<details>
<summary><b>Q3: 如何集成到现有项目？</b></summary>

**步骤**:

1. 添加依赖
2. 初始化 TracerProvider
3. 添加中间件（自动埋点）
4. 添加自定义 Span（手动埋点）
5. 配置 Collector
6. 部署监控平台

**详细说明**: [快速集成指南](#-5分钟快速开始)

</details>

<details>
<summary><b>Q4: 如何处理高基数属性？</b></summary>

```go
// ❌ 错误: 高基数属性
span.SetAttributes(
    attribute.String("user.email", email), // 基数太高
)

// ✅ 正确: 使用哈希或ID
span.SetAttributes(
    attribute.String("user.id", userID),
)
```

**详细说明**: [标签管理](./03_数据模型/02_Metrics数据模型/04_标签.md)

</details>

<summary><b>Q5: 日志如何关联追踪？</b></summary>

```go
// 使用 otelslog (自动关联)
handler := otelslog.NewHandler("myapp")
slog.SetDefault(slog.New(handler))

// Context 中自动包含 Trace ID
slog.InfoContext(ctx, "Processing request")
```

**详细说明**: [日志与追踪关联](./02_Semantic_Conventions/04_Log约定/03_日志与追踪关联.md)
