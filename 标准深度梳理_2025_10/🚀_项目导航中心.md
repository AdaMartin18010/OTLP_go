## 🛠️ 运行项目

### 环境要求

```bash
Go 1.25.1+
OpenTelemetry SDK v1.32.0+
Docker (可选，用于运行 Collector)
```

### 快速开始

```bash
# 1. 克隆项目
git clone <repository-url>
cd OTLP_go

# 2. 运行基础示例
cd examples/basic
go run main.go

# 3. 运行主项目
cd ../..
go run ./src/main.go

# 4. 运行 Docker 环境 (可选)
docker-compose up -d
```

### 验证安装

```bash
# 编译检查
go build ./...

# 代码质量检查
go vet ./...

# 格式检查
gofmt -l ./

# 运行测试
go test ./...
```
